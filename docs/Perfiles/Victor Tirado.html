Project overview

Hi, I am Víctor Tirado Fernández and during this project I have been contributing to the art team.

My main role has been asset creation for the levels, also I help to rig and animate, to create assets we followed a pipeline to get it. From 3DMax we created the low poly, after this we imported it to Zbrush to give details, and finally it was texturized in Substance Painter.
